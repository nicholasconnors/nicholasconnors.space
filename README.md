# nicholasconnors.space
My personal website for astrophysics studies.

Notes to self so I remember how to make websites:

1. Install [Node.js](https://nodejs.org/en/download). Ensure its properly added to the Path variable (Edit the system environment variables -> Environment variables -> Path -> Must include C:\Program Files\nodejs). This might require a restart.

2. Install [Astro](https://docs.astro.build/en/install-and-setup/).

3. Run locally for development with `npm run dev`.
